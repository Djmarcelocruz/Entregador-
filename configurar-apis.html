⚙️ Configurar APIs

Integração com serviços externos reais

💬 WhatsApp Business API

📌 Como obter:

1. Acesse developers.facebook.com

2. Crie um app e configure WhatsApp Business

3. Copie o Access Token e Phone Number ID

📞 Twilio SMS API

📌 Como obter:

1. Acesse twilio.com

2. Crie uma conta e configure SMS

3. Copie Account SID, Auth Token e seu número

🗺️ Google Maps API

📌 Como obter:

1. Acesse console.cloud.google.com

2. Crie um projeto e ative Google Maps API

3. Gere uma chave de API

💳 Stripe Payment API

📌 Como obter:

1. Acesse stripe.com

2. Crie uma conta e configure pagamentos

3. Copie suas chaves (Public e Secret)
